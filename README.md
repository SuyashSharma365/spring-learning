# Spring Learning

Practice project for learning **Spring Boot**, **REST API development**, and **MongoDB integration**.

## 🚀 Tech Stack

* Java
* Spring Boot
* Spring Web
* Spring Data MongoDB
* Maven

## 📂 Project Structure

```
src/main/java/com/suyash/springlearning
 ├── controller
 ├── service
 ├── repository
 └── model
```

## 📌 Features

* Build REST APIs using Spring Boot
* Connect Spring Boot with MongoDB
* Learn backend architecture (Controller → Service → Repository)
* Practice CRUD operations

## ⚙️ Running the Project

1. Clone the repository

```
git clone https://github.com/SuyashSharma365/spring-learning.git
```

2. Navigate to the project directory

```
cd spring-learning
```

3. Run the application

```
mvn spring-boot:run
```

The server will start at:

```
http://localhost:8080
```

## 📚 Purpose

This repository is used for learning backend development with Spring Boot and understanding how modern Java backend applications are structured.
